# 🚀 Featured Projects


## 🤖 Industrial Robotics Maintenance | KUKA KR6 R900

**ECU Robotics Lab**

- Performed fault diagnosis and recovery of a **KUKA KR6 R900** with **KRC4 Compact** controller.
- Diagnosed hardware failures including motherboard, KPP and KSP drive modules.
- Rebuilt the complete KUKA software environment (Windows, SmartPad, WorkVisual).
- Restored controller communication, safety configuration, and robot motion.
- Performed troubleshooting on E-Stop circuits, enabling switch logic, brake circuits, and power stages.
- Documented hardware/software root-cause analysis and corrective actions.

---

## 🦾 Upper-Limb Exoskeleton

**IEEE SSCS International Arduino Competition — 🥉 3rd Place Worldwide**

- Designed and developed a single-DOF powered upper-limb exoskeleton for rehabilitation assistance.
- Implemented high-torque actuation using an automotive window motor with a custom transmission mechanism.
- Developed the embedded control system using Arduino.
- Integrated mechanical and electrical safety constraints suitable for human interaction.

---

## 🤖 Autonomous Mobile Robot (AMR)

- Built an Autonomous Mobile Robot using ROS2.
- Developed robot models using URDF/Xacro.
- Worked with TF2, Navigation Stack, SLAM and robot localization.
- Implemented collision monitoring and autonomous navigation behaviors.
- Simulated and validated the robot using Gazebo and RViz.
  
---

## 🔦 Laser Audio Transmission System

- Developed a wireless audio transmission prototype using a laser beam and photovoltaic receiver.
- Designed the analog amplification stage for signal conditioning.
- Demonstrated point-to-point audio transmission between separate transmitter and receiver modules.
